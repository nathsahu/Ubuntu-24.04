 XFCE and KDE desktop . </p>

## Installation

<p> Paste this in termux </p>

```bash 
apt install wget -y 
wget https://raw.githubusercontent.com/23xvx/Termux-Ubuntu-Installer/main/Install.sh
bash Install.sh 
```
